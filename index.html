Game of Life Simulator

Instructions

Welcome to the Game of Life Simulator! To use the simulator, follow these steps:

  1. Click on the grid to add or remove cells.
  2. Use the 'Start' button to begin the simulation.
  3. Use the 'Stop' button to pause the simulation.
  4. Adjust the speed using the slider.
  5. Use the 'Reset' button to clear the grid.
  6. Explore different patterns by loading them from the dropdown menu.
